## 📌 项目展示

### 🔥 WotAI – WOT 坦克自动导航与辅助
**仓库地址：** https://github.com/liangyuR/wot_ai  

基于 Python + YOLO + 路径规划的游戏自动导航项目，包含：

- 小地图检测（YOLO）
- 敌我位置识别
- A* 路径规划 + cost map + 栅格膨胀
- 键盘鼠标控制器
- Debug overlay 可视化

> 项目在持续开发中，是目前主要的个人工程方向。
